# fl — image/video prompt guide

All generated media must sit inside the "craft desk" design system (docs/design.md): warm
paper and clay, editorial, tactile, calm. Never SaaS-blue, never neon, never glossy 3D.

Always include:

- palette: terracotta (#c75b39), cream/warm paper (#f6f1e7), espresso ink (#1b1611)
- light: soft diffused studio/morning light, gentle shadows
- texture: tactile, matte, grain, linen/paper/clay surfaces
- mood: calm, crafted, human hands at work
- camera: macro or close-up, shallow depth of field, slow drift if video

Avoid: text in image, faces in focus, saturated blues/purples, plastic/glass gloss, busy
backgrounds, fast motion.

Canonical subject family: hands making things — clay on a wheel, thread on a loom, plane
shavings, ink on paper. Work as a physical, warm act.
